# Простая Лента Постов

Простое одностраничное веб-приложение, которое отображает ленту постов. Пользователи могут добавлять новые посты и реагировать на них.

## Возможности

- Просмотр списка постов с изображениями и подписями.
- Добавление нового поста с указанием URL изображения и подписи.
- Реакции на посты с помощью эмодзи 'like' (нравится), 'wow' (ух ты) или 'laugh' (смешно).
- Базовая проверка форм и сообщения для пользователя.

## Стек Технологий

- **Фронтенд:** HTML5, CSS3, TypeScript
- **Сборка/Инструменты:** Node.js, npm, Компилятор TypeScript (tsc)
- **Тестирование:** Jest, ts-jest

## Структура Проекта

## Начало Работы

Следуйте этим инструкциям, чтобы запустить копию проекта на вашем локальном компьютере.

### Требования

Вам необходимо иметь установленные [Node.js](https://nodejs.org/) и [npm](https://www.npmjs.com/).

### Установка

1.  Склонируйте репозиторий или скачайте исходный код.
2.  Откройте терминал в корневой папке проекта.
3.  Установите зависимости проекта:
    ```sh
    npm install
    ```

### Сборка Проекта

Чтобы скомпилировать код TypeScript в JavaScript, выполните следующую команду. Результат будет помещен в директорию `public/js`.

```sh
npm run build
```

### Разработка

Для автоматической перекомпиляции кода TypeScript при внесении изменений, выполните:

```sh
npm run dev
```

### Запуск Приложения

Это статический веб-проект. После сборки проекта (`npm run build`), откройте файл `public/index.html` в вашем любимом веб-браузере.

## Тестирование

Для запуска автоматизированных тестов используйте следующую команду:

```sh
npm test
```
